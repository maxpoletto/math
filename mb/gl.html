Max iter limits the number of iterations used to determine whether the Mandelbrot sequence at each point is bounded. See what happens as the value approaches zero!

Hue base controls the display color. If a point is in the Mandelbrot set, its color is black. Otherwise, if it diverges after n iterations, its hue is computed as ((hueBase - n) mod 360).
